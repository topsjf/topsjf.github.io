  -->

                                              # 常用命令

                                              清空文件：mvn clean

                                              编译项目：mvn compile

                                              打包jar：mvn package

                                              打包并部署到本地maven目录里：mvn install

                                              .jar.original 结尾文件

                                              .jar.original 是普通 jar 包，不包含依赖

                                              .jar 是可执行 jar 包，包含了 pom 中的所有依赖，可以直接用 java -jar 命令执行

                                              如果是部署，就用 .jar

                                              如果是给别的项目用，就要给 .jar.original 这个包
